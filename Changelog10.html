Version 10.0.0 - 2012-08-24

  • New Features and general changes:
    • Renamed to UE Music Library.
    • Enable browsing by disconnected players over Comet and JSONRPC. Current Playlist management operations are redirected to the player's controlling server.
    • Add additional ports for Slim-discovery and HTTP access.
    • Add an operation mode whereby locally-connected players are not supported. Only new ports (above) used in this case.
    • Add an operation mode whereby only the local library is supported (no remote services).
    • Add an operation mode whereby ip3k-based players are not supported.
    • Support seeking for DLNA-type streaming for more stream format types: Ogg, MP4(AAC, ALAC), WMA
